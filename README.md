# 🌱 Krishaan Agro — Growing Together

Premium e-commerce & services website for **Krishaan Agro** — smart farming, urban gardening, contract farming, agri machinery, soil testing and premium agri products.

## ✨ Features

- 🎨 **Premium animated UI** — Framer Motion scroll reveals, parallax hero, ken-burns effects, count-up stats
- 🛒 **Cart + WhatsApp ordering** — enquiry-based pricing, COD, no payment gateway needed
- 🤖 **AI Agri-bot** — built-in chatbot for instant customer queries
- 🌿 **4 service verticals** — Krishaan Agro Services, Urban Gardening, Smart Farming, Contract Farming
- 📱 **Fully responsive** — mobile-first, works on every screen
- ⚡ **SEO-ready** — static generation (SSG) for all pages

## 🚀 Getting Started

```bash
# install dependencies
npm install

# run development server
npm run dev

# production build
npm run build && npm start
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## 🌐 Deploy

Easiest: [Vercel](https://vercel.com/new) — import this repo, zero config needed.

## 🛠️ Configuration

All contact details (phone, WhatsApp, email, address, socials) live in one place:
[`src/lib/utils.ts`](src/lib/utils.ts) → `SITE` object.

Products & services content: [`src/lib/data.ts`](src/lib/data.ts)
